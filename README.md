# 📖 Overview

DocumentGPT Pro is a locally running AI-powered document assistant that enables users to interact with PDF, DOCX and TXT documents using natural language.

Built using Retrieval-Augmented Generation (RAG), the application retrieves the most relevant document context through semantic search before generating responses with a locally hosted Large Language Model.

Unlike cloud-based AI assistants, DocumentGPT Pro performs document processing, embedding generation, vector search and language model inference entirely on the user's machine, ensuring complete privacy and offline accessibility.

The project demonstrates practical implementation of modern AI engineering concepts including vector databases, embeddings, prompt engineering, context construction and conversational document intelligence.

# ✨ Features

- 📄 Upload PDF, DOCX and TXT documents
- 🧠 AI-powered document summarization
- 💬 Conversational document question answering
- 🔍 Semantic similarity search
- 📚 Retrieval-Augmented Generation (RAG)
- 🤖 Local Llama 3.2 inference using Ollama
- 🔗 LangChain orchestration
- 🗄️ ChromaDB vector database
- 🧠 Hugging Face Sentence Transformers embeddings
- 📑 Source-grounded AI responses
- 📊 Context relevance scoring
- 📈 Document insights dashboard
- 💡 AI-generated suggested questions
- ⚡ Streaming responses
- 📤 Export conversations (TXT, Markdown and PDF)
- 🔒 100% Local & Privacy Focused
