## 🚀 Overview

DocumentGPT Pro is a production-ready AI-powered Retrieval-Augmented Generation (RAG) application that enables users to upload documents, build a semantic knowledge base, and interact with them using natural language.

The application runs completely offline using Ollama and combines LangChain, ChromaDB, Sentence Transformers, and Streamlit to deliver accurate document-grounded responses with source attribution.

---

## ✨ Features

- 📄 Upload PDF, DOCX and TXT documents
- 🧠 AI-generated document summaries
- 💡 Suggested questions generated automatically
- 🔍 Semantic search using vector embeddings
- ⚡ Retrieval-Augmented Generation (RAG)
- 🤖 Local Llama 3.2 inference with Ollama
- 📚 Source chunk visualization
- 📊 Document insights dashboard
- 🎯 Confidence scoring
- 💬 Multi-turn conversational memory
- 📤 Export chat as TXT, Markdown and PDF
- 🌙 Fully offline and privacy-focused
